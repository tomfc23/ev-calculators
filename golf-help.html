Golf help

iphone section

  1. Go to fanduel
  2. Select the golf event from the top sport select
  3. Hit "show all" for win only odds
  4. Take a screenshot of that
  5. Go into photos and copy and paste the text as you would usually do
  6. Copy and paste this text into the input box and follow directions on the site
  7. dm @tomfc on Real if you have any issues/questions

Other section (if the above does not work)

  1. Follow the steps above for taking a screenshot
  2. Attach this screenshot to Chatgpt and tell it to convert this image to text in the following format:"Enter players and odds (one per line, alternating name and odds)"
  3. Copy and paste this text into the input box and following directions on the site
  4. dm @tomfc on Real if you have any issues/questions
