How to find placement/Real Odds



  • For NYR, the placement for this is 3 since it has the 3rd highest percentage selected
  • If two percents are the same, rank it off the number in parentheses next to the percent
  • The real odds are the number next to the name. For NYR, the real odds are +100
  • Use a sportsbook such as Fanduel, Draft Kings, etc. for the sportsbook odds input
